This is a single page application made using Angular 9 for front-end development and Bootstrap as CSS framework.

For the API I used ASP.NET C# created with MVC pattern, as well as Entity Framework and LINQ to connect to my SQL Server database.

The app features: fully implemented messaging system between users, customizable user profiles including photo upload, ability to see other users as well as search for users based on age and gender, like feature and more!
